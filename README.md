# Renormind

**Renormind** 是一款极简的深度专注与任务管理工具，帮助你建立仪式感，重塑心流状态。

---

## ⚡️ 核心功能

1.  **CTDP 任务树**
    *   支持无限层级子任务，结构清晰。
    *   支持设定“计划时长”，自动计算偏差百分比。
2.  **神圣座位 (Sacred Seat)**
    *   独创的“预约 -> 专注”流程。
    *   倒计时结束后自动切换为任务正计时，无缝衔接。
3.  **后台保活**
    *   无论杀后台还是重启手机，计时器都能瞬间恢复，分秒不差。
4.  **完全离线**
    *   所有数据存储在本地，无需联网，保护隐私。

---
